# VerticalTextConverter
Convert text into vertical.

input sample:

颯颯西風…\
滿園栽\
蕊寒—\
香冷蝶難來﹏

output sample:

香蕊滿颯\
冷寒園颯\
蝶︱栽西\
難  風\
來  ︙\
︴   

using g++ to compile.\
still under construction.
